Follow the next steps in order to use correctly this plug-in:

Manual Entry:

  1. Insert User email.
  2. Create User temporary Password.
  3. Choose User Role (Merchant or User/Buyer).
  4. Click Add in order to add it to the list
  5. Repeat any times that is necessary.
  6. Click Create in order to execute the user creation (notification is send to all user email).

Using a CSV file:

  1. Insert User email.
  2. Create User temporary Password.
  3. Choose User Role (Merchant or User/Buyer).
  4. Click Add in order to add it to the list
  5. Repeat any times that is necessary.
  6. Click Create in order to execute the user creation (notification is send to all user email).
